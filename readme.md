Для установки темы загрузите папку Pervee в каталог themes Вашего Wordpress.

Скопируйте плагин helloanton.php из папки dev в дирректорию plugins Вашего Wordpress. Плагин добавит надпись Hello World! на главную страницу сайта, причем цвет сообщения будет меняться в зависимости от времени. На главной странице темы находится шорткод [hello_text], выводящий фразу 'Hello World'.
